About me

I'm David, a software engineer with a lifelong passion for all things tech. What started as a hobby at 15 has evolved into a career dedicated to creating impactful digital solutions. My toolkit is diverse: I'm fluent in Python, C#, Java, and PHP, and I build stunning user interfaces with HTML5 and CSS3. I also specialize in creating powerful and responsive single-page applications using React and ensuring type safety with TypeScript. I'm currently exploring new frontiers by integrating LLMs into my projects, creating smarter and more innovative user experiences. I also specialize in WordPress development, leveraging tools like Elementor and Bootstrap to create dynamic and responsive websites.
