# David Cit - Digital Portfolio & Business Website

[![Live Website](https://img.shields.io/badge/Live-Website-brightgreen?style=for-the-badge)](https://daiviis.github.io)
[![GitHub Pages](https://img.shields.io/badge/Deployed%20with-GitHub%20Pages-blue?style=for-the-badge&logo=github)](https://pages.github.com/)

A modern, minimalistic portfolio and business website showcasing digital solutions, web development services, and professional expertise.

## 🌟 Features

### 🎨 **Modern Design**
- **Minimalistic Interface**: Clean, professional aesthetic with strategic use of white space
- **Glass Morphism Effects**: Contemporary glass blur effects in navigation
- **Smooth Animations**: Subtle micro-interactions and transitions
- **Responsive Layout**: Optimized for all devices (mobile-first design)

### 🚀 **Performance Optimized**
- **Lightning Fast**: Optimized loading times and smooth animations
- **SEO Enhanced**: Comprehensive meta tags, structured data, and semantic HTML
- **Accessibility**: WCAG compliant with proper ARIA labels and keyboard navigation
- **Cross-browser Compatible**: Tested across all modern browsers

### 💼 **Business-Ready**
- **Contact Integration**: EmailJS powered contact form
- **Service Showcase**: Detailed pricing and service descriptions  
- **Portfolio Display**: Interactive project demonstrations
- **Conversion Research**: Data-driven optimization results
- **Legal Compliance**: Privacy policy and terms of service included

### 🛠 **Technical Stack**
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Styling**: Tailwind CSS with custom components
- **Typography**: Google Fonts (Oxygen family)
- **Animations**: CSS animations with intersection observers
- **Forms**: EmailJS integration for contact handling
- **Charts**: Chart.js for data visualization
- **Hosting**: GitHub Pages

## 🚀 Quick Start

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/daiViis/daiviis.github.io.git
   cd daiviis.github.io
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your preferred browser
   # Or use a local server:
   python -m http.server 8000
   # Or with Node.js:
   npx serve .
   ```

3. **Start developing**
   - Edit HTML files for content changes
   - Modify `script.js` for functionality updates
   - Customize styles in the `<style>` sections

### Deployment

The website is automatically deployed to GitHub Pages when changes are pushed to the `master` branch.

**Live URL**: [https://daiviis.github.io](https://daiviis.github.io)

## 🔧 Configuration

### EmailJS Setup
1. Create account at [EmailJS](https://www.emailjs.com/)
2. Set up email service and template
3. Update credentials in `script.js`:
   ```javascript
   emailjs.init("YOUR_PUBLIC_KEY");
   // Update service and template IDs
   ```

## 🎨 Design System

### Colors
- **Primary**: Black/White minimalistic palette
- **Accents**: Strategic green indicators for availability
- **Text**: Gray scale for hierarchy
- **Interactive**: Smooth hover transitions

### Typography
- **Primary Font**: Oxygen (Google Fonts)
- **Weights**: 300 (Light), 400 (Regular), 700 (Bold)
